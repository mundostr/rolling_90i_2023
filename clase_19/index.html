Primeros pasos con Javascript (JS)

Podemos insertar código js entre <script></script>, o bien utilizar el atributo src e indicar la ruta de un archivo js a importar.

Por una cuestión de orden, utilizaremos habitualmente la segunda opción.

Cambiar el nombre del archivo js para ir probando los distintos códigos.

Los operadores nos permiten ejecutar distintos tipos de acciones:

Asignación
El operador universal para asignar valor, por ej a una constante o variable.
  • = (igual)

Aritméticos
Son los operadores comunes que conocemos desde temprana edad, para cálculos matemáticos.
  • + (más)
  • - (menos)
  • * (multiplicación -> asterisco, NO x)
  • / (división)

Condicionales
Son los operadores de comparación, que aprovechamos cuando evaluamos condiciones (if())
  • > (mayor a)
  • >= (mayor o igual a)
  • < (menor a)
  • <= (menor o igual a)
  • === (igual a, se verifica valor y tipo)
  • !== (distinto a, se verifica valor y tipo)

Lógicos
Son los operadores booleanos con los que podemos enlazar condiciones
  • ! (not -> no -> negación)
  • && (and -> y)
  • || (or -> o)
